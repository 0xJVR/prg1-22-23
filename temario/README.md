|Windows|Linux|Mac
|-|-|-

### Carpetas: CRUD

| |Windows|Linux|Mac
|-|-|-|-
| Crear carpeta| **mkdir** *nombreDeCarpeta* <br> o <br> **md** *nombreDeCarpeta*| **mkdir** *nombreDeCarpeta*|**mkdir** *nombreDeCarpeta*
| Listar contenidos de carpeta| **dir**| **ls -l**| **ls -l**
